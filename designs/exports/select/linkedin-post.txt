Testing a shadcn/ui Select has a hidden challenge: the dropdown options render in a portal — outside the component tree.

Here's how to handle it:
→ Query the trigger using its combobox role
→ Click to open, then use screenDomGlobal to reach portal-rendered options
→ Select an option and verify the value updates

This is one of those things you only learn the hard way — or by seeing it in action. Swipe through for the full test code.
